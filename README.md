# üíß Hydro Kinetic

**Hydro Kinetic** is a modern, mobile-first web application designed to gamify daily water consumption. It features a realistic liquid wave engine that reacts to user input, wrapped in a high-end "Glassmorphism" interface.

[**View Live Demo**](https://hydro-kinetic-tracker.netlify.app) ## ‚ú® Key Features

* **Kinetic Visuals:** A realistic, physics-based liquid wave animation that rises as you hydrate.
* **Smart Tracking:** * Quick-add presets (Glass/Bottle).
    * Precision control (+/- custom amounts).
    * Visual progress percentage.
* **Gamification:**
    * **Celebration Engine:** Particle confetti explosion upon reaching daily goals.
    * **History & Streaks:** Visual 7-day calendar history with color-coded performance indicators (Red/Orange/Cyan).
* **Zero Dependencies:** Built entirely with vanilla HTML5, CSS3, and JavaScript. No frameworks or heavy libraries.
* **PWA Ready:** optimized meta tags allow for "Add to Home Screen" functionality on iOS and Android, providing a native app-like experience (full screen, no browser UI).
* **Privacy Focused:** Uses LocalStorage to persist data directly on the user's device. No external servers or data collection.
